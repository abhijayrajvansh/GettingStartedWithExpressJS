REST API - JSON

GET /users - HTML doc ✅
GET /api/users - list all users ✅
GET /api/users/1 - get the user data with id 1 ✅
GET /api/users/2 - get the user data with id 2 ✅

POST /api/users - create new user 

PATCH /api/users/1 - edit user with id 1

DELETE /api/users/2 - delete user with id 2


https://www.mockaroo.com/ for generating sample data
